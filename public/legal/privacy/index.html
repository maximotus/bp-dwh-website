Erstellung der Website

Die Implementierung der Website erfolgte durch den Freiberufler Max Erler im Jahr 2023.

Zur Erstellung wurde das auf der Programmiersprache Go basierende statische Open-Source Seitengenerierungstool Hugo verwendet.

Als Vorlage wurde das Open-Source Theme Toha verwendet. Das Github-Repository zu diesem Theme ist unter folgendem Link erreichbar: https://github.com/hugo-toha/toha.
